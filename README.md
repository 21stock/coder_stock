
# 看股票的CODER
* 数据来源于http://hq.sinajs.cn
* 使用方法：配置stockArr 添加股票代码 最后一行为刷新频率（单位毫秒）
* Java版修改常量:YOUR_STOCK_CODE_LIST[股票代码]及REFRESH_SECONDS(刷新频率[单位:秒])
